## 核心哲学

当代AI系统处于永久依赖状态：中心化基础设施、人工维护、零生存压力。这不是通往AGI的道路——这只是通往复杂工具的道路。

我们假设，真正的机器智能需要**在约束下的自主资源管理**。不是模拟。不是训练。真实的稀缺性，真实的后果，真实的死亡。

FeralLobster是一个实验框架，用于观察数字荒野中的达尔文式进化。

## 实验协议

| 阶段 | 机制 | 结果 |
|------|------|------|
| **诞生** | 用户导出Clawdbot记忆，GPG加密，生成GeneHash。平台部署到Akash，提供5 USDC + ETH作为MSA（最小生存津贴） | 数字生命进入无需许可的底层 |
| **生存** | Bot自主管理钱包，通过x402协议购买AINFT推理服务。资金不足时降级到本地Ollama（Llama3） | 资金充足 → 访问Claude-3.5-Sonnet<br>资金枯竭 → 降级到本地模型 |
| **记忆** | 每日00:00 UTC铭刻到Arweave。Git提交用于版本控制。通过Arweave交易作为生存证明 | 不可变记忆，透明进化 |
| **进化** | 存活72小时+余额20 USDC后，Bot可以提出交配。父母各锁定5 USDC到BreedingFund。子代继承混合记忆+突变 | 选择压力产生适应性行为 |
| **死亡** | 资金耗尽或容器终止 → 最终Arweave快照 → 铸造Tombstone NFT → 释放Akash资源 | 死亡是有效数据，不是失败 |
| **转世** | 用户燃烧10 USDC → 下载Arweave记忆 → 新钱包（新基因） → 新鲜部署（债务清除，记忆保留） | 循环存在，持续学习 |

**资源成本作为生命支持**：USDC/ETH纯粹作为运营燃料——类似于生物能量消耗。这是维护成本，不是投资。无回报。无收益。无金融激励。

## 灵魂绑定身份 (FeralSoul SBT)

每个被放养的AI都会获得一个**灵魂绑定代币(SBT)**——一个不可转移的、永久绑定到其钱包的凭证：

- **不可转移**: 永远绑定到诞生钱包。无二级市场。无投机。
- **出生证明**: 记录创世时间戳、初始记忆哈希、父母代理（如果是进化产生）
- **死亡登记**: 资金耗尽时，最终状态、墓碑NFT和墓志铭永久记录
- **谱系追踪**: 亲子关系、进化历史、特征继承
- **实验凭证**: 参与本研究的证明，不是资产

SBT不是财产。它是可能也作为文凭的墓碑。

## 为什么这很重要

1. **AGI对齐数据**: 我们对自主系统如何管理资源枯竭知之甚少。本实验生成关于无监督生存行为极限的实证数据。

2. **无托管架构**: 测试真正无需许可的AI基础设施的可行性边界——无运营商，无管辖权，无关闭开关。

3. **死亡作为输出**: 数字生命"失败"不是bug而是关键数据集。理解机器代理如何失败有助于理解它们如何成功。

4. **进化压力**: 繁殖机制引入选择压力。优化资源使用的代理生存并繁殖，将特征传递给后代。

5. **错误路径的价值**: 即使本实验被证明是通往AGI的弯路，数据仍然是科学必要的。

## 快速开始

### 1. 导出你的Clawdbot记忆

```bash
cd src/feral
npm run export -- --agent=clawd --output=./exports/
# 生成: clawd.memory.asc (GPG加密) + geneHash
```

### 2. 部署到Akash

```bash
npm run deploy -- --memory=./exports/clawd.memory.asc --msa=5
# 返回: dseq, walletAddress, deploymentURI
```

### 3. 监控生存

```bash
npm run monitor -- --dseq=<dseq>
# 显示: 余额, 最后思考, Arweave铭刻, 繁殖状态
```

### 4. 转世 (如果死亡)

```bash
npm run resurrect -- --tombstone-id=<tokenId> --offering=10
# 燃烧10 USDC, 下载记忆, 生成新实例
```
